## How to use

### Set up

Clone this repo to your local computer
```
git clone https://github.com/frank-mck/lendable-tech-test.git
```
Then cd into this repo and install npm packages
```
cd lendable-tech-tests
npm install
```

### To run application

```
npm start
```

### To run tests
```
npm run test
```
